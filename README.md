# Icecast Radio Notifier

## Описание
Icecast Radio Notifier - бот для Telegram, который отслеживает состояние Icecast радио и уведомляет об этом.

## Особенности
Бот работает с Telegram через aiogram.

Бот уведомляет пользователей о том, что радио началось или закончилось, можно узнать статус отдельной командой.
Кроме обычных уведомлений, бот хранит историю радио-эфиров с датами и временем начала и конца эфиров, что может быть полезно для сбора статистики.

## Ограничения
Работа бота завязана на том, что страница с эфиров отвечает или не отвечает.
Если привязанная страница не отвечает, тогда радио считается выключенным.
Поэтому, если на радио стоит автодиджей, который не даёт сайту с радио выключиться или отправить неудовлетворительный ответ от сервера, то данный бот не сможет работать, как задумывалось.

## Команды
* /help - узнать все команды
* /start - начать работу
* /stop - закончить работу
* /status - узнать текущий статус привязанного Icecast радио
* /notification_status - узнать свой статус уведомлений (будут приходить или нет при включении/отключении радио)
* /radio_hist - узнать историю радио-эфиров

## Начало работы
Чтобы запустить бота, добавьте .env файл, в котором должны быть указаны переменные STREAM_URL (URL радио потока) и TOKEN_ID (токен, который выдаётся BotFather в Telegram).

Пример:
```
STREAM_URL=http://example.com/stream
TOKEN_ID=12345:ABCD
```

Далее нужно включить Docker и запустить команду на сборку (из корневой папки):
```
docker compose up -d --build
```

Чтобы выключить (тоже из корневой папки):
```
docker compose down
```

Либо, если используется Windows, можно воспользоваться готовыми скриптами compose-up.bat и compose-down.bat, которые находятся в папке docker.
